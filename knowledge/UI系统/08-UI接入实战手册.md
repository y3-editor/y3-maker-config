## 1. 环境准备

### 1.1 键盘 / 鼠标同步（必须）

在 `global_script/global_main.lua` 的 `游戏-初始化` 事件里开启：

```lua
y3.game:event('游戏-初始化', function()
    y3.config.sync.key   = true   -- 键盘事件
    y3.config.sync.mouse = true   -- 鼠标事件
end)
```

> ⚠️ 在地图 `main.lua` 里设置无效，必须在 `global_main.lua` 里。

## 2. UI 节点获取（y3.ui.get_ui）

### 2.1 路径格式

| 格式 | 能否用 | 示例 |
|------|--------|------|
| `"layer名.节点名"` | ✅ 正确 | `'panel_1.layout_1'` |
| 裸节点 UID | ❌ 失败 | `'7a92193f-...'` |
| `"layer名/节点名"` | ❌ 失败 | `'panel_1/layout_1'` |
| Layer 名本身 | ✅ 可获取，但不能作为 prefab parent | `'panel_1'` |

### 2.2 调用时机

```lua
-- ❌ 错误：游戏-初始化同帧调用会 error
y3.game:event('游戏-初始化', function()
    local panel = y3.ui.get_ui(player, 'panel_1.layout_1')  -- nil / error
end)

-- ✅ 正确：延迟 1 帧
y3.game:event('游戏-初始化', function()
    y3.player.with_local(function(player)
        y3.ltimer.wait_frame(1, function()
            local panel = y3.ui.get_ui(player, 'panel_1.layout_1')  -- ok
        end)
    end)
end)
```

### 2.3 在 UI 编辑器找到正确路径

1. 在 Y3 UI 编辑器里打开目标 Layer（如 `panel_1`）
2. 点击目标节点，查看左下角「节点」面板的层级名
3. 路径 = `layer名.节点名`（多级：`layer名.父节点.子节点`）

## 3. UI Prefab 创建（y3.ui_prefab.create）

### 3.1 基本用法

```lua
---@param player Player
---@param prefab_name string  Prefab 名称（编辑器里的名字，不是 UID）
---@param parent_ui UI        必须是 Layer 内的节点，不能是 Layer 本身
---@return UIPrefab
local slot = y3.ui_prefab.create(player, 'artifactPickCmp', cached_panel)
```

### 3.2 子节点路径（get_child）

`UIPrefab:get_child(path)` 的起点是 prefab 内部的根 UI 节点：

```lua
-- artifactPickCmp 节点树：
-- root
-- └── bg (Button)
--     └── descr_TEXT (Label)

-- ✅ 正确（带 root. 前缀）
local bg   = slot:get_child('root.bg')
local desc = slot:get_child('root.bg.descr_TEXT')

-- ❌ 错误（少一层）
local bg   = slot:get_child('bg')           -- nil
local desc = slot:get_child('bg.descr_TEXT') -- nil
```

> **规律**：查看对应 `XxxCmp.lua` 源码里 `self._ui:get_child("root.xxx")` 的写法，prefab 内部子节点路径带 `root.` 前缀。

### 3.3 完整 Prefab 卡槽示例（含点击绑定）

```lua
local slot = y3.ui_prefab.create(player, 'artifactPickCmp', cached_panel)
if slot then
    -- 设置文字
    local desc = slot:get_child('root.bg.descr_TEXT')
    if desc then desc:set_text('攻击 +10') end

    -- 绑定点击
    local bg = slot:get_child('root.bg')
    if bg then
        bg:add_local_event('左键-点击', function()
            -- 处理选中逻辑
        end)
    end
end

-- 移除：slot:remove()
```

## 4. 键盘事件

### 4.1 事件类型

| 事件名 | 触发范围 | 适用场景 |
|--------|----------|---------|
| `'键盘-按下'` | 全玩家同步 | 多人同步操作 |
| `'本地-键盘-按下'` | 仅本地客户端 | 本地 UI 交互（推荐） |

### 4.2 KeyboardKey 常量速查

| 键类型 | 格式 | 示例 |
|--------|------|------|
| 字母键 | 单字母大写 | `'A'` `'F'` `'Z'` |
| 数字键 | `KEY_` + 数字 | `'KEY_1'` `'KEY_0'` |
| 功能键 | `F` + 数字 | `'F1'` `'F9'` `'F12'` |
| 特殊键 | 名字大写 | `'SPACE'` `'ENTER'` `'HOME'` `'ESC'` |

### 4.3 完整键盘事件接入示例

```lua
-- global_main.lua
y3.game:event('游戏-初始化', function()
    y3.config.sync.key = true   -- ⚠️ 必须，否则键盘事件不触发
end)

-- main.lua 或业务模块
y3.game:event('游戏-初始化', function()
    -- 本地按 F 键触发
    y3.game:event('本地-键盘-按下', y3.const.KeyboardKey['F'], function(trg, data)
        y3.player.with_local(function(player)
            -- 本地 UI 逻辑
        end)
    end)
end)
```

## 5. 常见陷阱总结

| 陷阱 | 症状 | 解法 |
|------|------|------|
| UID 传给 `get_ui` | `UI "xxx" 不存在` | 改为 `layer名.节点名` |
| 同帧 `get_ui` | `UI "xxx" 不存在` | `wait_frame(1, fn)` |
| `get_child('bg')` 找不到 | 返回 nil，点击无响应 | 加 `root.` 前缀 |
| `sync.key` 位置错 | 按键完全无反应 | 移到 `global_main.lua` |
| `KeyboardKey['KEY_F']` | 按键完全无反应 | 改为 `KeyboardKey['F']` |
| Layer 作为 Prefab parent | Prefab 创建失败 | 用 Layer 内的节点 |
| `export_ui` 超时 | 30s 超时 error | 先 `screenshot_ui` 唤醒 UI 编辑器 |
