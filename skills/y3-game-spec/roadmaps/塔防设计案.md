## 1. 游戏概述

经典 2.5D 俯视角塔防游戏。玩家选中基地后，从引擎建造面板挑选 4 种防御塔之一，在地图允许位置（碰撞通过即可）放置，抵御沿固定路径进攻的 10 波怪物。怪物到达终点会扣除基地生命，击杀怪物可获得金币用于建造和升级防御塔。成功击杀全部 10 波怪物（含最终 Boss）即为胜利，基地生命归零则失败。

**核心循环**：选基地→选塔技能→建塔 → 抵御怪物 → 获得金币 → 升级/建造更多塔 → 抵御更强波次 → 通关或基地被摧毁

---

## 2. 核心规则

| 规则 | 说明 |
|------|------|
| 游戏类型 | 单人经典塔防 |
| 核心循环 | 建塔 → 杀怪 → 获金币 → 强化防线 |
| 操作方式 | 默认锁定选中基地，引擎自动展示 4 个建造技能按钮；点击塔技能 → 引擎建造指示器 → 选位置确认建造；点击已建塔弹出升级/出售面板 |
| 胜利条件 | 击杀全部 10 波怪物（含第 10 波 Boss） |
| 失败条件 | 基地生命降为 0 |
| 波次机制 | 共 10 波，每 12 秒开始下一波（可配置 `Config.WAVE_INTERVAL`） |
| 怪物刷新间隔 | 同一波内每 1.2 秒刷一只（`Config.SPAWN_INTERVAL`） |
| 初始资源 | 金币 200、基地生命 10 |
| 建造方式 | 由真·建造技能（`ability_cast_type=4`）提供，无独立"建造区域"逻辑节点 |
| 选中锁定 | 基地常驻选中，禁用点选/框选操作（保证建造按钮始终可见） |

> ⚠️ **逻辑节点通过 MCP 静态摆放**（`add_point_path` 等），Lua 代码仅通过 `get_xxx_by_res_id` 引用，禁止代码动态创建。

| ID | 对象名 | Y3 类型 | 数量 | 功能 | 触发方式 |
|------|--------|---------|------|------|---------|

**路径航点规则**（详见 §7.3）：

| 规则项 | 说明 |
|--------|------|
| 路径类型 | 固定路径，14 个航点（编辑器 `res_id=10000`） |
| 移动方式 | `unit:move_along_road(road, '单向', false, true, false)` |
| 到达终点 | ltimer 0.5s 轮询单位坐标与终点距离，≤80 判到达 → 扣血 + 移除单位 |

### 3.5.3 地形效果规则

| 属性 | 值 |
|------|-----|
| 是否需要地形效果 | 不需要（纯视觉装饰） |
| 说明 | 经典塔防，地形不产生战斗效果。路径和建造区域通过纹理区分（路径=浅色地面纹理，建造区域=草地纹理），仅用于视觉引导，不影响单位属性或视野。 |

> 地形仅作视觉层。所有战斗效果（减速、AOE 等）通过防御塔技能/Buff 实现（见 §4.2 / §4.3），无需地形参与。
