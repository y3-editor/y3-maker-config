# Y3 地形纹理颜色映射表

> 来源：编辑器纹理面板实际截图 + texture-ids.md
> 用途：AI 在分配纹理时，根据原图颜色匹配最接近的纹理
> 
> **选择规则**：
> 1. 首先匹配颜色（主色调 RGB 相近）
> 2. 然后匹配材质语义（名称含义，如"雪地"不能选白色金属地面）
> 3. 同等颜色下优先选材质语义匹配的纹理

## ⚠️ AI 纹理选择警告

### 颜色近似但材质不同的易混淆组

| 白色/浅色 | ⚠️ 区分 |
|----------|---------|
| 90 雪地 (200,210,215) | 雪质感 → 仅用于雪景 |
| 160 白水泥地 (170,165,155) | 水泥质感 → 仅用于工业/人工场景 |
| 182 浅色路面 (185,175,165) | 路面质感 → 仅用于道路 |
| 181 白沙地 (190,180,160) | 沙子质感 → 用于海滩/白沙 |

| 灰色 | ⚠️ 区分 |

| 215 金属地面 (90,85,95) | 金属质感 → ⛔ 不用于自然地形 |
| 56 城区路面 (115,110,100) | 路面质感 → 城市区域 |
| 172 冻土 (95,100,105) | 泥土质感偏冷 → 寒冷区域 |

| 深绿色 | ⚠️ 区分 |
|--------|---------|

| 74 地底砖 (55,60,55) | 砖 → 地下城 |

### 核心原则
1. **颜色第一**：先按 RGB 距离找最近的 3~5 个候选
2. **材质第二**：从候选中选材质语义最匹配的
3. **场景第三**：考虑整体地图风格一致性
4. **禁止跨类**：自然地形不选人工材质（金属、水泥），雪地场景才选冰雪类
