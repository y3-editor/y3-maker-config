# Y3 地形纹理完整映射表

> 来源：`editor_terrain_texture.py` + `editor_export_language_data_zh.py`
> 共 170 种纹理，AI 在 Pass 2 写入纹理时按此表选择具体 `texture_type` ID。

## ⚠️ 地图纹理面板限制

Y3 编辑器中每个地图有一个**纹理面板**（最多 32 个纹理），只有面板中启用的纹理才能通过 MCP 写入。

**AI 通过 `ensure_terrain_textures` MCP 接口自动处理纹理面板**：
1. 在分配纹理后，调用 `ensure_terrain_textures(texture_ids=[...])` 自动将缺失纹理添加到面板
2. 返回值中 `added` 列表表示成功添加的纹理，`failed` 列表表示面板已满无法添加的纹理
3. 面板满时（32 个上限），AI 应将 `failed` 中的纹理替换为 `current` 列表中最接近的已有纹理
4. `not_downloaded` 列表中的纹理未下载到本地，需提示用户手动下载或选择替代纹理

## AI 选择纹理的指引

1. 根据图片色相和场景风格，从上面的分类中**选择具体的纹理 ID**
2. ⛔ 不要用笼统的 "grass" / "rock"，要选到具体 ID（如 #170 草地、#147 岩石）
3. 同一场景尽量用同系列纹理（如丛林区域全用 66~72 的丛林系列）
4. 纹理面板通过 `ensure_terrain_textures` MCP 接口自动管理，无需用户手动操作
